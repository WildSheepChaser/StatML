To run a generalization test make the two following folders:
- test-personal
- test-black-bg

  
The following folders should be filled with hand gestures representing rock paper scissors gestures made by the user or that belong to a different dataset than [this dataset](https://www.kaggle.com/datasets/drgfreeman/rockpaperscissors) , with homogenous backgrounds for test-black-bg (original images had a black background) and images with complex backgrounds (such as grid patterns) for test-personal
